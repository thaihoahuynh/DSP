The aim of this project is to display the phrase “Đại Học Quốc Gia – Trường ĐH Quốc Tế” on the first line of the screen followed by “DSP – 28Mar2026”on the secondline on the OLED LCD screen by using TMS320C5515 eZdsp USB Stick. In this project, we will not use the standard ASCII character libraries but rather the creation of a custom bitmap font. Utilize the “usbstk5515_BSL_RevA” given out to conduct the experiment.
